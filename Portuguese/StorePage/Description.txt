# Has to be < 300 characters
Bloody Rally Show é um jogo distópico de corrida de combate roguelite com infinita variedade de pistas, campanhas, missões, tabelas de líderes, desafios diários, editor de pistas, editor de carros, ajuste de carros, personalização, batalhas de carros e vários modos de jogo, incluindo um onde você é um pedestre.
